# Legit HUD

Este projeto contém um LocalScript em Lua para Roblox com:

- hotkey para ativar/desativar a HUD
- bind configurável para a tecla de toggle
- lista de jogadores com nome, equipe, vida, distância e visible check
- visible check via raycast para indicar se o alvo está visível

## Como usar

1. Coloque o arquivo [legit_hud.lua](legit_hud.lua) em um LocalScript dentro de StarterPlayer > StarterPlayerScripts.
2. Inicie o jogo para ver a HUD.
3. Use a tecla F para alternar a UI, ou clique no botão Bind para trocar a tecla.

## Observação

Este script foi pensado para uso legítimo em seu próprio jogo/servidor e não realiza autoaim nem altera a câmera.
