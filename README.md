# Legit HUD

Este projeto contém um LocalScript em Lua para Roblox com:

- overlay de HUD com painel de controle
- aimbot para mirar automaticamente no inimigo mais próximo visível
- silent aim que tenta ajustar o disparo de forma mais discreta
- bind configurável para a tecla de toggle
- lista de jogadores com nome, equipe, vida, distância e visible check

## Como usar

1. Coloque o arquivo [legit_hud.lua](legit_hud.lua) em um LocalScript dentro de StarterPlayer > StarterPlayerScripts.
2. Inicie o jogo para ver a HUD.
3. Use a tecla INSERT para alternar a overlay.
4. Use os botões no painel para ativar/desativar Aimbot e Silent Aim.

## Observação

Este script ajusta a câmera local para aimbot e silent aim. O comportamento exato depende do jogo e do sistema de disparo utilizado.
